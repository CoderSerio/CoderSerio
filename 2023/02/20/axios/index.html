概述

axios什么意思

axios 是 ajax i/o system的缩写，即”**异步 JavaScript 和 XML 输入输出 系统”

是一个基于promise网络请求库，作用域node.js和浏览器中。

前置准备

首先得建立一个vue项目，然后安装相关依赖

如果不清楚vue基础知识的话可以阅读我的博客：Vue全家桶学习笔记_零基础入门到入坑:Vue篇

这里我们采用的是vue-cli，语法是vue3.x

此处建议使用vue的GUI项目管理器进行创建，顺便把axios的相关依赖安装上（当然，使用npm或者yarn安装都行)

打开vue的GUI项目管理器只需要用cmd打开命令控制符然后输入下方的指令即可

vue ui

GUI的话就把依赖和插件都装上就行了

原生ajax示例

下面段代码就是原生js写的ajax请求，请求的内容是本地的json文件

httpRequest = new XMLHttpRequest();
           
if (!httpRequest) {  
    alert("创建请求失败");      
}

httpRequest.open("GET", "./JSONtest.json");
           
httpRequest.send();

httpRequest.onreadystatechange = function () {       
    if (httpRequest.readyState === 4) {         
        if (httpRequest.status === 200) {  
            var data = JSON.parse(httpRequest.responseText);              
            console.log('AJAX' + data['key']);
                        //别忘了和data.key的区别             
        }          
        else {              
            console.error('请求失败');            
        }          
    }          
}

使用axios

axios封装了很多通信方式

名称
request
get
post
delete
head
options
put
patch

get请求

以下两种写法都是可以的

(url随便给的，赋值粘贴之后记得换成后端的接口，或者本地的文件路径之类的)

const axiosGetTest = () => {
        axios.get('http://rap2api.taobao.org/xxx').then((res) => {
          console.log(res.data)
        }).catch((error) => {
          console.log(error)
        })
    }

    const axiosGetTest2 = () => {
      axios({
        method: 'get',
        url: 'http://rap2api.taobao.org/xxx',
      }).then((res) => {
        console.log(res.data)
      }).catch((error) => {
        console.log(error)
      })
    }

get方法还可以写第二个参数，那就时配置信息

可以按住ctrl+鼠标左键点击get方法看源码，再如此操作第二个参数，就可以查看能够填写哪些内容作为第二个参数

这里我们加一个params，就可以实现传参了

const axiosGetTest = () => {
        axios.get('http://rap2api.taobao.org/app/mock/292586/user/list',{
          params: {
            username: '千尊',
            id: 666
          }
        }).then((res) => {
          console.log(res.data)
        }).catch((error) => {
          console.log(error)
        })
    }

这里似乎没有用utf-8所以中文部分乱码了

get请求就是以这种拼接的形式传递数据

post请求

依旧是两种，post和get相比就多了中间一个参数data

const axiosPostTest = () => {
      axios({
          method: 'post',
          url:'http://rap2api.taobao.org/yyy',
          data: {
            "isOk": false,
            "errMsg": "失败了诶1"
          }
      }).then((res) => {
        console.log(res.data)
      }).catch((error) => {
        console.log(error)
      })
    }

    const axiosPostTest2 = () => {
      axios.post(
          'http://rap2api.taobao.org/yyy',
          {
            "isOk": false,
            "errMsg": "失败了诶2"
          },
          {}).then((res) => {
        console.log(res.data)
      }).catch((error) => {
        console.log(error)
      })
    }

信息是以请求头+请求体的形式发送的

当然还有其他通信方式，它们的专长不一样，但是使用方式类似，这里就不再赘述了

并发请求

什么是并发？

如果不清楚的话可以参考我放在文章开头的操作系统笔记，系统地学习并发并行异步等专业术语

如果知道什么是并发的话，那么请接着往下看

并发请求要用到axios.all方法，其返回结果是一个数组，可以用axios.spread将数组的值展开

const axiosAll =  () => {
  axios.all([
      axios.get('http://rap2api.taobao.org/app/mock/292586/user/list'),
      axios.post('http://rap2api.taobao.org/app/mock/292586/user/login')
  ]).then((res) => {
    for(let i of res){
      console.log(i.data);
    }
  }).catch((error) => {
    console.log(error)
  })
}

至于axios.spread方法，感觉很多时候没什么用，还不如for循环，所以这里就不再演示了
